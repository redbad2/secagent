## 样本哈希分析 SOP

分析文件哈希（MD5/SHA1/SHA256），获取样本的恶意行为、传播路径、关联域名/IP。

---

### 步骤 1：沙箱分析

调用 qianxin_fdp_sample__sandbox（entity_type="sha1"）

**获取信息：** 样本类型、文件大小、检测率、行为标签、通信 IP/域名、下载 URL

**判断规则：**
- 检测率 > 50% → 高度恶意
- 检测率 10%-50% → 可疑，需进一步分析
- 检测率 < 10% → 可能是误报或新样本
- 有 C2 通信记录 → 高风险
- 有下载 URL → 关联分析该 URL

---

### 步骤 2：关联 IP/域名分析

从沙箱结果中提取通信 IP 和域名，逐个查询威胁情报：

- 调用 secagent__ip_threat 查询通信 IP
- 调用 secagent__domain_threat 查询关联域名
- 调用 secagent__ip_attacks 查询 IP 攻击链

**判断规则：**
- 关联 IP 有 C2 标签 → 确认恶意
- 关联域名有恶意标签 → 确认恶意
- 关联 IP/域名均为白名单 → 可能是合法软件

---

### 步骤 3：IOC 提取

从分析结果中提取 IOC：
- 样本哈希（MD5/SHA1/SHA256）
- C2 IP/域名
- 下载 URL
- 文件名

---

### 步骤 4：综合评估

**评分框架：**
- 沙箱检测率 > 50% + 有 C2 通信 → 严重
- 沙箱检测率 > 30% + 有恶意关联 → 高
- 沙箱检测率 10-30% → 中
- 沙箱检测率 < 10% → 低

**输出：**
- 风险等级、置信度、恶意家族（如有）、IOC 列表、关联基础设施
